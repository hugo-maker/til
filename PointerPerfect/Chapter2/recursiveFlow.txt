1 -> 2 -> 3 -> print
                 |
     <------------
     3 -> 2 -> print
                 |
<----------------- 
2 -> 1 -> 3 -> print
                 |
     <------------
     3 -> 1 -> print
                 |
<----------------- 
3 -> 1 -> 2 -> print
                 |
     <------------
     2 -> 1 -> print


再帰トレースの際は、関数を呼び出した位置(=帰ってくる位置) に何らかのマーキングを施しておく(イロハとか)
